# Квалификационное задание для команд из учебных групп

## Содержание квалификационного задания

1. Развернуть инфраструктуру для разработки

2. Запустить проект, выполнить тесты (если применимо)

3. Создать по шаблону необходимые папки и файлы для своего проекта

4. Создать новый тип сообщений, передать с использованием брокера между сервисами 

5. Написать автотест и оценить уровень покрытия кода тестами

6. Задокументировать своё решение в соответствующем документе в формате md

7. Записать экран и прислать запись с результатом выполнения задачи
