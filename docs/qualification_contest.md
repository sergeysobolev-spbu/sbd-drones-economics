# Квалификационное задание

## Содержание квалификационного задания

1. По заданным ЦБ и архитектуре оценить уровни доверия доменов безопасности

2. Написать автотест и оценить уровень покрытия кода тестами

3. Выполнить статический анализ кода из примера, исправить замечания, повторно выполнить проверку и убедиться, что замечаний больше нет

4. Используя пример, получить сертификат для прошивки

5. Используя пример, получить сертификат для дрона

6. Используя пример, зарегистрировать дрон в платформе агрегатора

7. Используя пример, получить и выполнить тестовое задание без киберпрепятствий
