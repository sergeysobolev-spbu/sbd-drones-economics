# Концепция развития (учебного) проекта

Проект моделирования экономики эксплуатации безопасных дронов может развиваться в двух направлениях:

| Направление | Описание | Особенности |
| --- | --- | --- |
| 1. Концепт технологии (КТ) | акцент на добавление новых функций и сценариев, связка с оборудованием и инженерными соревнованиями | новые функции, высокая изменчивость, НИР формат, возможен выход на НИОКР
| 2. Обучающий проект (ОП) | акцент на обучении студентов курсов, связанных с проектированием и разработкой систем с конструктивной информационной безопасностью | лабораторные работы, контрольные задачи, акцент на шаблоны проектирования, сравнительно мало изменений год-к-году, главным образом добавление новых упражнений, связь с УМК |

Это могут быть разные репозитории, код между которыми может переиспользоваться в обоих направлениях.

## Векторы развития направлений

| Направление | Описание | Особенности |
| --- | --- | --- |
| КТ | - интеграция с различными НУС <br> - интеграция с симуляторами БАС <br> - внедрение технологий ИИ <br> - внедрение логики роевого/группового выполнения задачи <br> - внедрение перспективных протоколов взаимодействия <br> - внедрение киберпрепятствий и сценариев соревнований <br> - поддержка новых аппаратных платформ |
| ОП | - добавление новых упражнений на разработку защищённого ПО <br> - внедрение новых шаблонов проектирования СКИБ <br> - создание сквозных автоматизированных тестов <br> - формальная верификация отдельных свойств безопасности <br> - внедрение метрик и механизмов их сбора для автоматизированной оценки работ студентов <br> - внедрение элементов геймификации в проект <br> - написание качественной технической документации по проекту с использованием ИИ <br> - применение мультиагентных технологий ИИ для генерации качественного кода (при наличии технической возможности, например, если будут доступны российские ресурсы)

## Целевая аудитория проекта

Основные пользователи обоих направлений - студенты и преподаватели. 

Но если формат обучающего проекта предполагается использовать на потоках (будут предложены лабораторные работы, предполагающие командное выполнение), концепт будет использоваться теми студентами, которые заинтересованы в более глубоком погружении в тематику СКИБ, отрасль БАС (автономных роботов в целом) и видят для себя перспективы научного и/или профессионального развития в этой области. Т.е., например, для написания дипломных работ.

В целом оба проекта предполагают открытость к доработкам и привлечению сообщества.

### Связь с учебно-методическими комплексами по теме СКИБ

Существует вариант УМК, построенный на существующих учебных примерах, он передаётся ВУЗам. 
Есть активность по созданию новой версии УМК, нацеленной как на ИБ, так и на ИТ направления обучения. 
Новая версия УМК может иметь свой набор лабораторных работ, а может использоваться совместно с проектом (по желанию преподавателя?).
